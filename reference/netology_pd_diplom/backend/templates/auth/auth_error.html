Возможные причины:

  • Неверный CLIENT_ID или CLIENT_SECRET
  • Несовпадение redirect_uri в настройках OAuth App
  • Истёк срок действия кода авторизации (~10 минут)
  • Пользователь отменил доступ в диалоге провайдера
  • Отсутствует scope user:email для GitHub

Проверьте настройки:

GitHub: Settings → Developer settings → OAuth Apps

Google: console.cloud.google.com → APIs & Services → Credentials

🔍 Логи Django: запустите сервер с флагом --verbosity 2
